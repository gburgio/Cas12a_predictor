1. Install dependencies (tested versions shown):
   - Python 3.8
     - scikit-learn==1.2.1
     - pandas==1.5.3
     - joblib==1.2.0


2. Add sequences to sequences.txt
   - One sequence per line
   - In the format of NNNNTTTNNNNNNNNNNNNNNNNNNNNNNNNNNN

3. Run the predictor using "python c12a_predictor.py"
